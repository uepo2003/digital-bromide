# クイックスタートガイド ⚡

5分で電子ブロマイド配布システムを使い始める！

## 📋 前提条件

- Node.js 18以上がインストールされている
- npmまたはyarnが使える
- Supabaseアカウント（無料）

## 🚀 3ステップでセットアップ

### ステップ 1: Supabaseプロジェクトを作成（5分）

1. **[Supabase](https://supabase.com)** にアクセスしてサインアップ
2. 「New Project」をクリック
3. プロジェクト名を入力（例: `digital-bromide`）
4. パスワードを設定（安全な場所に保存）
5. リージョンを「Northeast Asia (Tokyo)」に設定
6. 「Create new project」をクリックして待つ（約2分）

### ステップ 2: データベースをセットアップ（2分）

1. 左サイドバーの **「SQL Editor」** をクリック
2. 「New Query」をクリック
3. プロジェクトの `supabase/schema.sql` ファイルを開く
4. **全ての内容をコピー**してSQL Editorにペースト
5. **「Run」** をクリック
6. 成功メッセージが表示されればOK！

### ステップ 3: アプリを起動（1分）

```bash
# 1. パッケージをインストール
npm install

# 2. 環境変数ファイルを作成
cp .env.example .env.local

# 3. .env.localを編集
# Supabaseの Project Settings > API から以下をコピー:
# - Project URL → NEXT_PUBLIC_SUPABASE_URL
# - anon public key → NEXT_PUBLIC_SUPABASE_ANON_KEY

# 4. 環境変数をチェック
npm run check-env

# 5. 開発サーバーを起動
npm run dev
```

🎉 **完了！** http://localhost:3000 にアクセスしてください

## 📱 初回の使い方

### 1️⃣ ブロマイドを作成

1. `http://localhost:3000/admin` にアクセス
2. 右上の **「新規作成」** ボタンをクリック
3. 画像をドラッグ&ドロップ（またはクリックして選択）
4. 情報を入力:
   - **タイトル**: 例「東江茜 電子ブロマイド No.001」
   - **シリアルナンバー**: 例「001」
   - **説明**: 任意
5. **「作成」** ボタンをクリック

✅ ブロマイドが一覧に表示されます！

### 2️⃣ ユーザーを作成

1. 上部メニューの **「ユーザー管理」** をクリック
2. 右上の **「新規ユーザー」** ボタンをクリック
3. 情報を入力:
   - **名前**: 例「山田太郎」
   - **メールアドレス**: 例「yamada@example.com」
   - **パスワード**: 自動生成されます（🔑アイコンで再生成可能）
4. **パスワードをメモ**してください（重要！）
5. **「作成」** ボタンをクリック

✅ ユーザーが作成されました！

### 3️⃣ ブロマイドを割り当て

1. ユーザーカードの **「ブロマイド割当」** ボタンをクリック
2. ドロップダウンから配布するブロマイドを選択
3. **「割り当て」** ボタンをクリック
4. 生成されたアクセスリンクの横にある **📋 コピー** ボタンをクリック

✅ リンクとパスワードをユーザーに送信しましょう！

### 4️⃣ ユーザーが受け取る

ユーザーに以下を送信:
```
━━━━━━━━━━━━━━━━━━━━━
🎁 電子ブロマイドが届きました！
━━━━━━━━━━━━━━━━━━━━━

アクセスリンク:
https://your-domain.com/receive/xxxxx...

パスワード:
XXXXXXXXXXXX

上記リンクにアクセスして、
パスワードを入力してください。
━━━━━━━━━━━━━━━━━━━━━
```

ユーザーがリンクを開くと:
1. パスワード入力画面が表示される
2. パスワードを入力
3. ブロマイドが表示される
4. ダウンロードボタンで保存できる

✅ 配布完了！

## 💡 よくある質問

### Q: 画像はどこに保存されますか？
A: 現在はBase64エンコードでデータベースに保存されています。本番環境ではSupabase Storageの使用を推奨します。

### Q: パスワードを忘れた場合は？
A: 現在はパスワードリセット機能がないため、新しいユーザーを作成してブロマイドを再割り当てしてください。

### Q: 1人のユーザーに複数のブロマイドを配布できますか？
A: はい！同じユーザーに複数回「ブロマイド割当」を実行してください。

### Q: アクセスリンクの有効期限はありますか？
A: 現在は無期限です。将来のバージョンで有効期限機能を追加予定です。
